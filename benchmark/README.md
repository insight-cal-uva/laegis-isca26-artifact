This repository contains the codebase for benchmarking NVIDIA H100 Confidential Computing (HCC), which aims to characterize HCC performance on the University of Virginia's servers.

This benchmark was initially taken from [UVM_benchmark](https://github.com/OSU-STARLAB/UVM_benchmark).

We modified some of the scripts and code to support *nsys* and HCC.
